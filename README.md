# Portfolio Website

A modern portfolio website featuring a 3D circular slider and multiple interactive sections.

## Features

- **3D Portfolio Slider**: Interactive circular slider with 11 portfolio cards using GSAP
- **Hero Section**: Animated landing area with engaging visuals
- **About Section**: Personal information and background
- **Services Section**: Services offered with interactive elements
- **Video Section**: Video content integration
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## Technologies Used

- HTML5
- CSS3 (with 3D transforms and animations)
- JavaScript (ES6+)
- GSAP (GreenSock Animation Platform)
- Modern CSS Grid and Flexbox

## Project Structure

```
htm/
├── index.html          # Main homepage
├── index.css           # Global styles
├── index.js            # Main JavaScript
└── sections/
    ├── about/          # About section files
    ├── hero/           # Hero section files
    ├── portfolio/      # Portfolio 3D slider
    ├── services/       # Services section
    └── video/          # Video section
```

## Portfolio Slider Features

- 3D circular arrangement of portfolio cards
- Smooth GSAP animations and transitions
- Card expansion with detailed information
- Click-outside-to-close functionality
- Touch/drag navigation support
- Keyboard navigation
- Mobile-responsive design

## Setup & Usage

1. Clone this repository
2. Open `index.html` in your browser
3. Navigate through different sections
4. Interact with the 3D portfolio slider

## Browser Support

- Modern browsers with CSS3 transform support
- Mobile browsers with touch events
- Requires JavaScript enabled
